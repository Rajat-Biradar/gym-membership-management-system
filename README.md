# Gym Membership Management System

## Overview

The Gym Membership Management System is a full-stack web application developed to manage gym members, memberships, attendance tracking, and renewal requests.

The system provides separate dashboards for administrators and members with secure JWT-based authentication and role-based access control.

---

## Features

### User Features

* User Registration
* User Login
* Profile Management
* Membership Details
* Attendance Tracking
* Attendance History
* Membership Renewal Requests

### Admin Features

* Admin Dashboard
* Member Management
* Membership Management
* Attendance Monitoring
* Renewal Request Approval and Rejection
* Cascade User Deletion
